# 🧮 Calculadora Bhaskara

> 🎓 Projeto de prática — desenvolvido durante meus estudos de Python.

Calculadora de equações do 2º grau (ax² + bx + c = 0) simples.

## ✨ Funcionalidades

- Calcula Delta (discriminante)
- Retorna x₁ e x₂
- Permite múltiplos cálculos em loop
